# Editor settings

The Editor settings provides a new centralized configuration panel that allows you to customize editor behavior at an individual user level. It provide greater flexibility, consistency, and control during the authoring process.

This centralized settings panel enables you to manage key editor preferences from a single location, reducing the need for scattered or manual configurations. These settings directly influence the authoring experience.

![Editor settings](./images/conf-editor-settings.png)

## Supported configuration options

You can enable or disable the following options based on your preferences:

![Editor settings](./images/editor-settings-dialog.png)

- **Non breaking spaces**: Controls the usage and visibility of non-breaking spaces while authoring content.
- **XML comments**: Enables the visibility of XML comments to reduce visual clutter or aid in structured authoring.
- **Tags**: Configure how XML/DITA tags are displayed in the Editor,with attributes or without attributes
- **Quick insert menu in the Editor**: The Editor Settings also include options to customize the Quick Insert menu that is used to configure and manage favorite elements for quick access during authoring in Editor.
You can search and add the frequently used elements in the Favorites of the Quick insert menu. The elements added here are displayed in the Editor when you press Control + / for Windows, Command + / for Mac.
