# December 2025 release of Product Learning and Training Content  

This release note covers the new and enhanced features introduced in December 2025 release of Product Learning and Training Content.

- **New options in the Insert menu**: Introducing new options features in the Editor toolbar to enrich your learning content: 

    - Math Equation: Insert MathML equations seamlessly for technical or scientific topics.
    - Knowledge Check: Add quick, non-graded quizzes to to validate learner understanding.
    - H5P Integration: Embed interactive H5P packages for an enhanced learning experiences.

    For more details, view [Other options in the Insert menu](../learning-content/lc-other-insert-options.md).
- Reusable content: You can now reuse existing content across multiple courses. This feature helps maintain consistency and reduces duplication.
- New interactive widgets: Now, you can add new interactive widgets to learning content and engage learners with interactive elements that make content more immersive:

    - Click to reveal
    - Filp card
    - Tabs

    For details, view [Use interactive widgets](../learning-content/lc-widgets.md).
- Create review task: Now, you can create a review task for your learning course and assign it to Reviewer(s) for their feedback. 

    For details, view [](../learning-content/manage-course.md#create-review-task).

- **Match the following**: A new question type, Match the Following, is available for quizzes. Now, learners can match items from two lists to form correct pairs, giving them an enhanced learning experience. 

    For details, view [Quiz question types](../learning-content/quiz-insert-questions.md#question-types).
