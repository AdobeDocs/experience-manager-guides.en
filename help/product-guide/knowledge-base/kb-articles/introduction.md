# Experience Manager Guides Knowledge Base

The Knowledge Base contains practical, scenario-driven articles for Experience Manager Guides covering common configurations, customizations, and use cases beyond the core product documentation.

| Category | Description |
|---|---|
| Authoring | Tips and techniques for creating and reusing content in the Editor, including editor customizations, metadata management, and content validation. |
| Guides Extension Framework | Learn how to customize the Experience Manager Guides UI from getting started with the extension framework to building custom components, widgets, and Review App customizations. |
| External Data Source | Guidance on integrating and configuring external data source connectors with Experience Manager Guides. |
| Publishing | Best practices for output generation, including Native PDF, DITA-OT, AEM Sites templates, and microservice-based publishing configurations. |
| Workflows | Configure and use post-generation workflows to automate steps after output is produced. |
| Translation | Instructions on how to improve translation performance and manage what content is included or excluded from translation. |
| Use cases | Learn about the real-world scenarios like conditional content, working with keys, and contextual content variables in published output. |
