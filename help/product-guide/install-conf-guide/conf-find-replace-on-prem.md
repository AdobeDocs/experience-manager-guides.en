# Configure Find and replace (Source view) for On-Premise

The following steps explain how to enable the Find and replace (Source view) in your On-Premise environment.

1.  Open the Adobe Experience Manager Web Console Configuration page.

    The default URL to access the configuration page is:

    ```http
    http://<server name>:<port>/system/console/configMgr
    ```

1.  Search for and select the **com.adobe.fmdita.config.ConfigManager** bundle.

1.  Enable the setting **Enable markup find and replace** (enable.markup.findreplace).  

1.  Select **Save**.
