# Customize Editor {#id181AI0003PN}

AEM Guides comes bundled with a powerful Web Editor that allows your authors to create and edit DITA documents. You can customize the  Editor's toolbar to add or remove any feature that you can access from the toolbar. Also, you can configure auto-savings of files, generating IDs for elements as they are inserted in your documents, and much more.
