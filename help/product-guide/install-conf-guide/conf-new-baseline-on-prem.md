# Configure New Baseline for On-Premise

The following steps explain how to enable the New Baseline in your On-Premise environment.

1.  Open the Adobe Experience Manager Web Console Configuration page.

    The default URL to access the configuration page is:

    ```http
    http://<server name>:<port>/system/console/configMgr
    ```

1.  Search for and select the **com.adobe.fmdita.xmleditor.config.XmlEditorConfig** bundle.

1.  Enable the setting **Enable faster Baseline (v2)**.  

1.  Select **Save**.
