# Configure New map collection for On-Premise

The following steps explain how to enable the New map collection in your On-Premise environment.

1.  Open the Adobe Experience Manager Web Console Configuration page.

    The default URL to access the configuration page is:

    ```http
    http://<server name>:<port>/system/console/configMgr
    ```

1.  Search for and select the **com.adobe.fmdita.config.ConfigManager** bundle.

1.  Enable the setting **Enable new map collections** (enable.new.map.collections).  

    ![](assets/new-map-collection.png){width="800"}

1.  Select **Save**.
